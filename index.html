My Apps and Websites

Math Quiz Game

This project made by me is actualy a multiplayer game based on the popular subject
Maths. I made this fun game with three coding languages - HTML, CSS and JS!
In this game you first have to Login with two names as this is a multiplayer game,
then you will be taken to a whole different page where the game is. each player
will keep asking and answering questoins, each correct answer will add 1 point to
your total score.

Hand Gesture Recognizer

This project i made is also based on the three coding languages - HTML, Java Script and
CSS. This web app identifies the gesture you make with your hand, but it is only limited to
three gestures - Peace, Ok and Amazing. You just have to look at the webcam view and make the
Gesture with your hand, Once You are satisfied with the gesture you can click on the button next
to it and then bellow the image you captured, there is a button to identify the gesture
which you have to press to identiy the gesture!

Animal Sound Recognizer

This project is based on neural networks and voice recognition. And like always, I
Used the three coding languages - HTML, Java script and CSS to make this! You just have
Click on yellow start button and then play the sound of either a dog, a cat or a
lion and then the respective image should respond while the other images dissapear!
